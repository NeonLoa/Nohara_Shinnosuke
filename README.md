# 野原新之助 桌宠

蜡笔小新桌面宠物 — Electron 透明悬浮窗，陪你工作摸鱼的小新。

## 功能

- **置顶悬浮** — 透明无边框，始终在最上层，不占任务栏
- **单击互动** — 跳一下 + 随机台词气泡（时间感知，早上说早安、晚上犯困）
- **双击** — 特殊动画（转圈 + 动感光波）
- **拖拽移动** — 自由拖到屏幕任意位置
- **自动游走** — 每隔一段时间在桌面上散步，碰到边缘反弹
- **打瞌睡** — 闲置 3 分钟后自动犯困
- **系统托盘** — 显示/隐藏、设置、退出
- **台词自定义** — 设置面板中增删改台词库

## 启动

```sh
npm install
npm start
```

## 设置

右键宠物 → 设置，可调整：

- 宠物大小（0.4x ~ 3x）
- 移动速度
- 气泡显示时长
- 窗口透明度
- 台词库编辑

## 技术栈

- Electron 35
- Vanilla HTML/CSS/JS
- 无框架依赖
